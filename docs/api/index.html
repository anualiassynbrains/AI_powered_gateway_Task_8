🧠 AI Summarization API

📍 Endpoint

POST /summarize

🔐 Headers

X-API-Key: your-api-key
Content-Type: application/json

📤 Request Body

JSON with one field:

{
  "text": "Your input text to summarize"
}

📥 Response

Returns the summary and credit usage info:

{
  "summary": "Concise summary of the input text.",
  "cost": 4,
  "remaining_credits": 96
}

⚠️ Error Responses

  • 429 Too Many Requests – If not enough credits.
  • 404 Not Found – If input is missing or summarization fails.

💡 Notes

  • Each request consumes 1–10 credits based on text complexity.
  • User balances are tracked in Redis using their API key.
